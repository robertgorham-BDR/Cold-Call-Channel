# Call Activity Log

**Purpose:** Daily/weekly call tracking. Rob fills this in during or after each calling session. The weekly-analytics task reads this every Friday and the system-diagnostics skill reads it every Sunday.

**DB:** `analytics/outreach.db` → `call_activity` table (auto-synced by weekly-analytics task)

---

## How to Log Calls

After any calling session, paste this template and fill it in:

```
### [YYYY-MM-DD] — [Morning/Afternoon/Evening]
- Dials: [N]
- Connects: [N]  (picked up, not voicemail)
- Voicemails: [N]
- Meetings booked: [N]
- Best time slot: [time range if you noticed a pattern]
- Notable connects:
  - [Name @ Company] — [outcome: not interested / follow-up scheduled / wrong person / pitched]
  - [Name @ Company] — [outcome]
```

**Quick log (minimum viable entry):**
```
### [DATE]
- Dials: [N] | Connects: [N] | Meetings: [N]
```

**Tip:** Even logging just Dials/Connects/Meetings takes 30 seconds and gives the system enough data to compute your connect rate and surface coaching insights automatically.
